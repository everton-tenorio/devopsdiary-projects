Welcome to the API

This API provides endpoints for user registration, login, and management. Below is a list of available endpoints to get you started.

POST /auth/register

Register a new user account.

Request Body: JSON with username and password.

Example: {"username": "user", "password": "pass123"}

Response: Returns a success message and user ID on successful registration.

POST /auth/login

Log in to an existing user account.

Request Body: JSON with username and password.

Example: {"username": "user", "password": "pass123"}

Response: Returns a JWT token for authentication.

GET /users

List all registered users.

Authentication: Requires a valid JWT token in the Authorization header (Bearer token).

Example: Authorization: Bearer

Response: Returns a list of users.

Note: You must be authenticated to access this endpoint.

GET /users/me

Retrieve information about the authenticated user.

Authentication: Requires a valid JWT token in the Authorization header (Bearer token).

Example: Authorization: Bearer

Response: Returns the authenticated user's details.

Note: You must be authenticated to access this endpoint.

To interact with the API, start by registering a user at /auth/register, then log in at /auth/login to obtain a JWT token. Use this token to access protected endpoints.
